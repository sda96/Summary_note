| -------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |

### 통계학습의 분류 - 모수적 방법
모델을 f를 구하는 것이 통계학습의 목표로써 모델f를 특정 모델이라고 가정하여 모수를 추정하는 방법을 모수적 방법이라고 합니다.

모수적 방법의 특징
- 선택한 특정 가정을 가진 모델이 모델f와 적합하지 않는 경우가 있음
- 데이터의 개수가 적을 수록 신뢰성이 떨어짐  

<img src = 'https://imgur.com/bC9w7FE.jpg' width=400 height=500 />

### 통계학습의 분류 - 비모수적 방법
모델 f의 형태에 대해 명시적인 가정이 없는 상태로 f를 추정하는 방법이 비모수적 방법입니다.

비모수적 방법의 특징
- 모수적 방법에 비해서 f의 형태를 적합되어질 가능성이 높음
- 모수적 방법에 비해서 훨씬 많은 데이터가 요구되어짐
